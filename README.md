## Url Shortening Service
This project is a short url service project from [roadmap.sh](https://roadmap.sh/projects).

## Project Setup
Run this project using run <kbd>docker-composer up</kbd> command
You can verify the application health on [lando](localhost:8080/health-check)

## Api
If you want to run this project on postman, you can import the postman collection inside the <kdb>./postman</kdb> directory.
